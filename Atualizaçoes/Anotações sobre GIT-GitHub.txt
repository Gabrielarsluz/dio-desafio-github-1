Após modificar algo no repósitório local:

- git status =  mostra aas novas modificações;

- git add . = adiciona o o arquivo modificado no controle de versão LOCAL.

- git -A = adiciona o o arquivo modificado no controle de versão LOCAL.
em qlqr pasta.

- git commit -m ( o -m permite adicionar um comentário ao commit) = prepara a modificação para ser enviada para a nuvem.
